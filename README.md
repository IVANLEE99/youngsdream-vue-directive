# YoungsDream Vue Directive

Enterprise-grade Vue2 + Vue3 directive ecosystem.

## Features

- Vue2 + Vue3 dual support
- Monorepo architecture
- pnpm workspace
- Changesets auto versioning
- GitHub Actions auto publish
- Pro private package ready

## Installation

```bash
pnpm install
pnpm build
```

## Commercial Model

- Core: MIT open-source
- Pro: Paid license
- Enterprise customization available
